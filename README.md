## Run Locally

Clone the project

```shell
  git clone https://github.com/01001111o1/CMSC128_BackEnd.git
```

Go to the project directory

```shell
  cd CMSC128_BackEnd
```

Install dependencies

```shell
  pip install -r requirements.txt
```

Start the server

```shell
  python run.py
```
