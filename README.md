## Run Locally

Clone the project

```bash
  git clone https://github.com/01001111o1/CMSC128_BackEnd.git
```

Go to the project directory

```bash
  cd CMSC128_BackEnd
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  npm run start
```
